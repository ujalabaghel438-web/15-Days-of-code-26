# 👋 Welcome to Vision CSE – 15 Days of Code Challenge 🚀

This repository is part of the **15 Days of Code Challenge** organized by **Vision CSE** to build coding consistency, strengthen problem-solving skills, and track daily learning progress.

---

## 📌 About the Challenge

- **Duration:** 2 Phases × 15 Days  
- **Goal:** Build the habit of coding every day  
- **Objective:** Learn, practice, and document progress consistently  

---

## ✅ How to Participate

1. Fork this repository  
2. Update your `README.md` in your fork  
3. Document your daily progress by adding:
   - Topics you learned
   - Problem-solving practice
   - Screenshots of your work
   - Project or submission links
4. Commit and push your updates daily
