# 👋 Welcome to Vision CSE – 15 Days of Code Challenge 🚀

This repository is part of the **15 Days of Code Challenge** organized by **Vision CSE** to help students build coding consistency, improve problem-solving skills, and track daily progress.

---

## 📌 About the Challenge

- **Duration:** 2 Phases × 15 Days  
- **Goal:** Build the habit of coding every day  
- **Objective:** Learn, practice, and document progress consistently  

---

## ✅ How to Participate

1. Fork this repository  
2. Update this `README.md` file in your fork  
3. Document your daily progress by adding:
   - What you learned or practiced
   - Screenshots of your work (optional)
   - Links to your code submissions/projects
4. Commit and push your progress daily
